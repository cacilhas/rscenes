# Rscenes

A scenes manager for [Raylib][] 4.5.

## Use

Add to your `Cargo.toml`:

```toml
[dependencies.rscenes]
version = "1.0"
git = "https://github.com/cacilhas/rscenes"
branch = "1.0"
```
