# Rscenes 2

Rscenes 2 is a game framework over [Raylib][], it was fully rewritten from
scratch for the release 2.0.

Only shades (2D) are tested, models (3D) are still being tested.

## Installation

```sh
cargo add rscenes
```

See [Nonogram][] and the [examples][] for use.
